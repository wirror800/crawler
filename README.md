# crawler(Python+Scrapy+Redis)
基于scrapy的网页爬虫

# 说明
本项目基于scrapy实现多线程爬取网页内容。

# 依赖环境
Python2.7

# 安装其他依赖及工具
        $ pip install cffi
        $ pip install libffi-dev
        $ pip install cryptography
        $ pip install mysql-python
        $ pip install service_identity
        $ pip install pypinyin
        $ pip install redis

# 使用方法
> $ scrapy crawl flat
